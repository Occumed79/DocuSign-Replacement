# Security Operations Dashboard

## Goal

Provide centralized visibility into platform integrity, fraud risk, and security operations.

## Planned Widgets

- tamper alerts
- verification failures
- anomaly heatmap
- high-risk signing queue
- WebAuthn adoption metrics
- SIEM delivery status
- artifact finalization status
- integrity verification status

## Planned Operational Views

### Fraud Review Queue

Review:
- suspicious signings
- high anomaly scores
- risky IP behavior
- integrity mismatches

### Integrity Monitoring

Track:
- evidence verification success rate
- finalized artifact integrity
- manifest generation
- audit export activity

### Security Telemetry

Track:
- OTEL events
- alert volume
- verification traffic
- auth escalation events
