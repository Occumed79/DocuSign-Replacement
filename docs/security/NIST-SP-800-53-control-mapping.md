# NIST SP 800-53 Control Mapping

## AC - Access Control

- Route-level authorization
- Public/private endpoint separation
- Ordered signing enforcement

## AU - Audit and Accountability

- Signature audit logging
- PDF download logging
- Decline/reminder event logging
- Evidence-hash tracking

## IA - Identification and Authentication

- Password hashing
- Session token hashing
- Electronic records consent enforcement

## SC - System and Communications Protection

- CSP protections
- Helmet security headers
- HTTPS-only deployment guidance

## SI - System and Information Integrity

- Dependency scanning
- CodeQL static analysis
- Secret scanning
- Evidence hashing

## RA - Risk Assessment

- EPSS prioritization guidance
- CISA KEV review process
