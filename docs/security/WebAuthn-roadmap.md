# WebAuthn Roadmap

## Goal

Add phishing-resistant authentication and high-assurance signer verification.

## Planned Capabilities

- Hardware security key support
- Passkey support
- Biometric-backed authentication
- Strong signer identity verification

## Candidate Libraries

- @simplewebauthn/server
- @simplewebauthn/browser

## Planned Use Cases

- Administrative MFA
- High-risk signing workflows
- High-value document approvals
- Sensitive PHI workflows

## Future Signing Enhancements

- WebAuthn-backed signing attestations
- Cryptographic signer attestation storage
