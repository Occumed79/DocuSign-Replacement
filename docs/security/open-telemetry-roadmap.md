# OpenTelemetry Roadmap

## Goal

Provide structured observability and security telemetry.

## Planned Telemetry

- request tracing
- signing-flow tracing
- evidence verification tracing
- database latency metrics
- security-event telemetry
- anomaly telemetry

## Candidate Exporters

- OTLP HTTP
- Datadog
- Grafana Tempo
- Jaeger
- Honeycomb

## Planned Benefits

- operational debugging
- fraud investigation
- security analytics
- performance monitoring
