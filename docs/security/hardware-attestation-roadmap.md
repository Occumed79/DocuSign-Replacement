# Hardware Attestation Roadmap

## Goal

Increase signer identity assurance using hardware-backed authentication.

## Planned Technologies

- WebAuthn passkeys
- FIDO2 security keys
- TPM-backed attestations
- Secure enclave-backed signing

## Planned Benefits

- phishing resistance
- stronger signer attribution
- reduced credential theft risk
- stronger non-repudiation

## Future Possibilities

- device trust scoring
- enterprise-managed keys
- biometric-backed signing
